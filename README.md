### **Installation**

1. Clone repository
   ```bash
   git clone https://github.com/ProjectPlantCapstoneRamalKriptoID/backend
   cd backend
   ```
2. Install dependencies
   ```bash
   npm install
   ```

### **Running App**

- **Development**:

  ```bash
    npm run dev
  ```

<!-- - **Production**:
  ```bash
